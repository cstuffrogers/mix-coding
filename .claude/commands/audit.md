# /audit — 全面项目健康检查

## 触发方式
```
> /audit
```

## 执行流程
1. 全量安全扫描（XSS/SQL注入/CSRF/硬编码密钥/不安全依赖）
2. 五层代码审查（ESLint → react-doctor → Playwright → AI语义 → 聚合）
3. 依赖安全审计
4. 性能静态分析（热点函数/大Bundle/N+1查询）
5. 测试覆盖率获取（仅报告）
6. 代码复杂度扫描（圈复杂度/重复率/文件行数/依赖深度）
7. 汇总所有检查结果为统一健康报告
8. 生成审计报告（docs/audit-report.md）
9. 交互式确认是否执行自动修复
10. 自动修复（可选）
11. 成本报告
12. 通知完成

## 说明
一键全量项目体检，整合 review + hunt + analyze 的能力。生成统一健康报告，可选择自动修复发现的问题。
