# /onboard — 开发者入职/环境搭建

## 触发方式
```
> /onboard
```

## 执行流程
1. 检测项目语言生态和所需工具链
2. 检查系统是否安装了必需的运行时和工具
3. 安装项目依赖（自动选择包管理器）
4. 检查.env配置，提示缺失的环境变量
5. 交互式填写缺失的环境变量
6. 生成.env文件
7. 开发构建验证
8. 运行测试验证环境
9. 启动开发服务器（后台）
10. 验证开发服务器正常启动
11. 记录环境搭建信息到项目记忆
12. 通知完成

## 说明
一键环境搭建，新人clone项目后运行即可。自动检测语言生态、安装依赖、配置环境变量、验证构建和启动开发服务器。
